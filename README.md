## 🏃 How to Run
1. Clone the repository  
   ```sh
   git clone [https://github.com/your-repo.git](https://github.com/ewfx/gaipl-ai-chargers.git)
   ```
2. Install dependencies  
   ```sh
   cd code
   npm install
   (for Python)
   cd code/server
   pip install -r requirements.txt 
   ```
3. Run the project  
   ```sh
   cd code
   npm start

   (For python backend)
   cd code/server
   python app.py
   ```
